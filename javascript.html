Variable Declaration

Declaring variables using "let".This snippet demonstrates how to declare a variable using the"let" keyword. The variable "x" is assigned a value of "10" and logged to the console. Variables declared with "let" are block-scoped and can be updated later.

        console.log(x);

Array Methods

Filtering array elements. This snippet demonstrates the "filter" method, which creates a new array with elements that satisfy the given condition. In this case, it filters numbers greater than 3.


        const numbers = [1, 2, 3, 4, 5];
        const filtered = numbers.filter(num => num > 3);
        console.log(filtered);

Array Transformation

Mapping array elements. This snippet demonstrates the "map" method, which creates a new array by applying a function to each element. It doubles the numbers in the array.


        const numbers = [1, 2, 3, 4, 5];
        const doubled = numbers.map(num => num * 2);
        console.log(doubled);

Spread Operator

Expanding array elemnts. This snippet demonstrates the spread operator, which allows copying elements from one array into another.

Destructuring

Extracting values from arrays and objects.This snippet demonstrates destructuring assignment, which extracts specific values into variables.

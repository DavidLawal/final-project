CSS ELEMENTS

Basic Styling

This demonstrates basic styling for a paragraph element, setting font size and color.


        p {
            font-size: 16px;
            color: #333;
        }

Background Color

This changes the background color of a div element.


        div {
            background-color: lightblue;
        }

Text Alignment

This aligns text to the center of an element.


        h1 {
            text-align: center;
        }

Box Shadow

This adds a box shadow effect to a div element.


        div {
            box-shadow: 5px 5px 10px rgba(0, 0, 0, 0.1);
        }

Font Styling

This changes the font family and size of a paragraph.


        p {
            font-family: 'Arial', sans-serif;
            font-size: 14px;
        }

Hover Effect

This changes the background color of a button when hovered over.


        button:hover {
            background-color: #4CAF50;
        }

Border Radius

This rounds the corners of a div element.


        div {
            border-radius: 10px;
        }

Padding and Margin

This adds padding inside an element and margin outside of it.


        div {
            padding: 20px;
            margin: 10px;
        }

Font Weight

This sets the font weight to bold for an h2 element.


        h2 {
            font-weight: bold;
        }

Flexbox

This uses flexbox to align items in a container.


        .container {
            display: flex;
            justify-content: space-between;
            align-items: center;
        }

Grid Layout

This uses CSS Grid to create a two-column layout.


        .container {
            display: grid;
            grid-template-columns: 1fr 1fr;
        }

Opacity

This sets the opacity of an image to 0.5, making it semi-transparent.


        img {
            opacity: 0.5;
        }

Text Shadow

This adds a shadow effect to the text of an h1 element.


        h1 {
            text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.3);
        }

Positioning

This positions a div element absolutely within its parent.


        .child {
            position: absolute;
            top: 10px;
            left: 20px;
        }

Text Decoration

This underlines text in a paragraph element.


        p {
            text-decoration: underline;
        }

Overflow

This controls overflow behavior for an element with a fixed height.


        div {
            height: 100px;
            overflow: scroll;
        }
